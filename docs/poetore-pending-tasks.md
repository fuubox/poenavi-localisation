## P0: 利用価値・検索可否に直接関わる機能

### 未鑑定Uniqueの画像付き候補カード

- [ ] 同一ベースの候補を、アイコン画像＋Unique名のカード一覧で表示する
- [ ] 候補が1種類なら自動選択し、複数の場合だけ選択画面を表示する
- [ ] 選択後は`identified=false`を含む正しい検索条件を送る
- [ ] Legacy等の同名Variantを候補内でどう区別するか確認する
- [ ] 画像データの取得元・キャッシュ・第三者ライセンス・配布サイズを確認する

v2.6.0では誤検索防止のため、未鑑定Uniqueを読み取ると非対応案内を表示し、検索ボタンを無効化している。

### Valdo Mapの報酬検索

- [ ] 実物の詳細コピー、内部クエリ、日本語公式Tradeへ渡すクエリを並べて再調査する
- [ ] 報酬option ID、type、discriminator、Foil条件、日英名称変換を確認する
- [ ] Completion Reward完全一致、Foil、実Mod、Void死亡Mod除外をAwakened準拠で組み立てる
- [ ] APIが受理するだけでなく、日本語公式Trade画面へ報酬条件が復元されることをWindows実機で確認する

v2.6.0では報酬条件検索が非対応であることを表示し、報酬条件を除いて検索する。
Valdo固有Modの解析・Trade stat解決は実装済み。
